# Images publiques

Ce dossier contient les images statiques utilisées dans l'application.

## Avatar de l'assistant IA

L'image `ai-assistant-avatar.png` est utilisée comme avatar du bot dans le Nuply Matching.

**Emplacement requis :** `/public/images/ai-assistant-avatar.png`

**Spécifications recommandées :**
- Format : PNG (avec transparence) ou JPG
- Taille : 256x256px minimum (sera recadrée en rond)
- Ratio : 1:1 (carré)
- Poids : < 100KB recommandé

L'image sera automatiquement recadrée en cercle dans l'interface. Assurez-vous que le sujet principal (visage) soit centré pour un meilleur rendu.
