# Thumbnail de la vidéo hero

## Fichier requis

### `hero-video-thumbnail.jpg`

**Spécifications :**
- Format : JPG ou WebP
- Ratio : 16:9 (ex: 1920x1080 ou 1280x720)
- Taille : < 200KB (optimisé pour web)
- Contenu recommandé :
  - Screenshot du dashboard en arrière-plan
  - Play button bien visible au centre (sera ajouté par le composant)
  - Texte overlay optionnel : "Démo en 45 secondes"
  - Style moderne et attrayant

**Outils recommandés :**
- Figma (création du design)
- Photoshop/GIMP (édition)
- TinyPNG (compression)

## Fallback

Si le fichier n'existe pas, un placeholder avec gradient sera automatiquement affiché.
